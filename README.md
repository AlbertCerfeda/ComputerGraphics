### Rendering
1. `cd` into the target assignment folder
```bash
cd Assignment1
```
2. Run the script
```bash
./render.sh
```
**Note:** if you add the `--open` flag the rendered image will be opened by your default application for `.png` files.
```bash
./render.sh --open
```
